# India EV Market Analysis – Power BI Dashboard

This project analyzes India's Electric Vehicle (EV) market using Power BI.

Key Insights:
- Total Vehicles Analyzed: 10K
- Battery EV Share: 70%
- Hybrid EV Share: 29%
- Average Range: 193 km

Dashboard Features:
- EV adoption trend over time
- Market split between BEV and PHEV
- Top EV brands by vehicle count
- State-wise EV distribution

Tools Used:
- Power BI
- Excel
- Data Visualization
